# Machine Learning Classification Project

This project demonstrates a basic machine learning workflow using a Kaggle dataset.

## Algorithms Used
- Decision Tree
- Random Forest

## Steps Followed
1. Dataset loading
2. Data cleaning and preprocessing
3. Feature encoding
4. Train-test split
5. Model training
6. Accuracy comparison

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn

## Output
The project compares the accuracy of Decision Tree and Random Forest models.
